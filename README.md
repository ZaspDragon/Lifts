# Lift Inventory Tracker

Separate GitHub Pages app for tracking lift inventory.

## Colors
- Green = Owned
- Yellow = Rental

## Files
- `index.html`
- `style.css`
- `app.js`

## Deploy
1. Create a public GitHub repo.
2. Upload these files.
3. Go to Settings > Pages.
4. Set source to `main` and folder to `/root`.
5. Save.

Your app link will look like:

`https://YOURUSERNAME.github.io/lift-inventory-github-app/`
